# 时政

# 党务知识 “二三四五六”

# 和党有关的 “二”

# 1. 两个维护

2018 年 8 月 26 日中共中央在新修订的《中国共产党纪律处分条例》中增写：

坚决维护习近平总书记党中央的核心、全党的核心地位

坚决维护党中央权威和集中统一领导。

# 2. 两个巩固

2013 年 8 月 19 日习近平总书记在全国宣传思想工作会议上提出：

巩固马克思主义思想在意识形态领域的指导地位

巩固全党全国人民团结奋斗的共同思想基础

# 3. 两学一做

2016 年 2 月中共中央办公厅印发《关于在全体党员中开展 “学党章党规、学系列讲话，做合格党员” 学习教育方案》中提出：

学党章党规

学系列讲话

做合格党员

# 和党有关的 “三”

# 1. 党的三大作风

毛泽东同志在党的七大所作的《论联合政府》的报告中提出：

理论联系实际

密切联系群众

批评与自我批评

# 2. 党的三大法宝

1939 年 10 月毛泽东同志在《〈共产党人〉发刊词》中提出：

统一战线

武装斗争

党的建设

# 3. “三会一课”

十三届三中全会上提出：

定期召开支部党员大会、支部委员会、党小组会

按时上好党课

# 和党有关的 “四”

# 1. 四个自信

2016 年 7 月 1 日习近平总书记在庆祝中国共产党成立 95 周年大会上的讲话提出：

道路自信

理论自信

制度自信

文化自信

# 2. 四个意识

2016 年 1 月 29 日中共中央政治局会议提出：

政治意识

大局意识

核心意识

看齐意识

# 3. 党员的四个合格

2017 年 3 月底中共中央办公厅在《关于推进 [“两学一做”](# 3. 两学一做) 学习教育常态化制度化的意见》中提出：

政治合格

品德合格

执纪合格

发挥作用合格

# 4. 四讲四有

[“两学一做”](# 3. 两学一做) 学习教育中提出：

讲政治 有信念

讲规矩 有纪律

讲道德 有品行

讲奉献 有作为

# 5. 四个服从

六届六中全会上首先提出：

个人服从组织

少数服从多数

下级服从上级

全党服从中央

# 6. 党面临的 “四大危险”

胡锦涛同志继 2011 年 “七一” 讲话之后再次向全党发出警示，并且首次写入党代会报告：

精神懈怠的危险

能力不足的危险

脱离群众的危险

消极腐败的危险

# 7. 党面临的 “四种考验”

党的十七届四中全会明确提出：

长期执政考验

改革开放考验

市场经济考验

外部环境考验

# 8. 党的 “四自” 能力

十八大中提出：

自我净化

自我完善

自我革新

自我提高

# 9. 党内政治生活 “四性”

2016 年 7 月 1 日习近平总书记在庆祝中国共产党成立 95 周年大会上的重要讲话中提出：

政治性

时代性

原则性

战斗性

# 和党有关的 “五”

# 1. “五位一体” 总体布局

十八大中提出：

经济建设

政治建设

文化建设

社会建设

生态文明建设

# 2. 党的五种纪律处分

《中国共产党纪律处分条例》中提出：

警告

严重警告

撤销党内职务

留党察看

开除党籍

# 和党有关的 “六”

# 1. 党的六大纪律

十八届五中全会前最后一次政治局会议上提出：

政治纪律

组织纪律

廉洁纪律

群众纪律

工作纪律

生活纪律

# 2. “六有” 政治局面

1957 年 7 月毛泽东同志提出：

有集中又有民主、有纪律又有自由、有统一的意志又有个人心情舒畅生动活泼的政治局面
